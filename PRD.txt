# **1. Product Overview**

This website is for a small boutique business. Customers will scan a QR code and land on a mobile-friendly website where they can:

* Browse categories of clothing (Designer Blouses, Dresses, Short Frocks, Long Frocks, Lehengas, Kids Wear, etc.)
* View designs inside each category
* View detailed information for each design
* Add items to a local cart (no backend)
* Suggest their own custom design via WhatsApp
* Schedule a store visit via WhatsApp
* Contact the boutique through WhatsApp or by phone

**Admin (you) can update designs manually by editing a `designs.json` file and redeploying on Vercel.**

---

# **2. Goals & Non-Goals**

### **2.1 Goals**

* Create a simple, beautiful, responsive boutique website.
* Host for free with zero backend costs.
* Allow customers to:

  * View catalogue categories
  * Browse designs
  * View details
  * Send enquiries via WhatsApp
  * Send custom design suggestions via WhatsApp
  * Schedule boutique visits via WhatsApp
* Keep development fast, simple, and lightweight.

### **2.2 Non-Goals (For Phase 1)**

* No admin dashboard
* No real database
* No image uploads from the website
* No login/authentication
* No payments or online ordering
* No API backend (Spring Boot, Node, or Firebase)

---

# **3. Target Users**

### **End Customers**

* Boutique customers scanning the QR code
* Mostly mobile users
* Looking for designs quickly and easily

### **Admin / Boutique Owner**

* Will update designs manually by editing the JSON dataset
